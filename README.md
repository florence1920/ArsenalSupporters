# ArsenalSupporters

## ✨ Concept
아스날 서포터 페이지

## 🎁 Contents
#### a. main
  - 경기 일정 (날짜, 경기장, 상대)
  - goal, assist 기록
  - best 11 명단
#### b. matches
  - 경기 정보
  - 최근 상대 전적
  - 응원 댓글 기능
#### c. team
  - 전체 선수 명단

## 🔨 Stack
1. html5, css3
2. vue cli 3.2.1
3. javascript


## ✔ How to run (on VScode)
1. 디렉토리 asnalspp
2. npm i @vue/cli@3.2.1
3. npm run serve
